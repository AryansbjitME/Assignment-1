HTML Notes

HTML Basics

HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages. Some HTML elements have no content. These elements are called empty elements. Empty elements do not have an end tag!HTML is a markup language that defines the structure of your content.

HTML Tags

HTML documents are made up of HTML tags. Tags are used to mark up elements on a webpage. Some common tags include <p> for paragraphs, <a> for links, and <img> for images.There is a range of HTML tags, they help you to design your web page. There are four required tags in HTML.

HTML Attributes

HTML attributes provide additional information about HTML elements. Attributes are specified within the opening tag of an element and consist of a name and a value. For example, the src attribute specifies the source URL of an image in the <img> tag.All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag

HTML Headings

HTML provides six levels of headings, from <h1> to <h6>. Headings are used to structure and organize the content on a webpage.HTML defines six levels of headings. A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading.
